# HearthPhone

A smartphone addon for World of Warcraft. Gives your character a fully functional phone with apps, games, messaging, and social features — all inside the game.

Type `/phone` to open it.

## What's Inside

### Communication
- **Messages** — Guild, Party, Raid, Instance chat and whispers in one unified inbox
- **Phone** — Ring a friend and they get a popup on their phone
- **Social Network** — Twitter/Reddit-style feed with posts, comments, emoji reactions, @mentions, and profiles. Syncs with other HearthPhone users on your realm and faction

### Games
- **Snake** — Classic snake
- **Tetris** — Falling block puzzle
- **Tic-Tac-Toe** — Multiplayer, challenge friends
- **Battleship** — Multiplayer naval warfare
- **Candy Crush** — Match-3 puzzle
- **2048** — Sliding number tiles
- **Minesweeper** — Flag the mines
- **Flappy Bird** — Tap to fly through pipes
- **Wordle** — Daily word guessing
- **Angry Birds** — Slingshot physics
- **Space Invaders** — Retro arcade shooter
- **Temple Run** — Endless runner
- **Subway Surfers** — Lane-dodge runner
- **Agar.io** — Multiplayer blob eating (syncs with players on your realm and faction)
- **Survivor** — Top-down roguelike shooter with WASD movement, mouse aiming, waves, loot power-ups
- **Tower Defense** — Grid-based TD with 3 tower types, upgrades, AoE splash, and escalating difficulty

### Tools
- **Calculator** — Basic math
- **Calendar** — In-game calendar view
- **Timer** — Stopwatch and countdown
- **Notes** — Write and save notes
- **Weather** — Current zone weather
- **Camera** — Screenshot tool
- **DPS Meter** — Personal DPS/HPS, group leaderboards, and Death Recap with timestamps, overkill, and absorbed damage
- **Fitness Tracker** — Steps, distance, activity, route planning on the world map
- **Music Player** — Play custom MP3s from your addon folder
- **Uber** — Flight map using hearthstones
- **Gallery** — Browse and set custom wallpapers

## Features

- Draggable phone with realistic bezel
- Lock screen with PIN code
- Multi-page home screen with swipe navigation
- App reordering with iOS-style edit mode
- Status bar with clock, gold, and zone
- Notification banners with click-to-open
- Settings for display, sound, security, and wallpapers
- Multiplayer games and social features sync via addon channels
- Cross-realm friend calling and presence detection via BNet
- All data saved across sessions

## Custom Content

**Music:** Drop `.mp3` files in `HearthPhone/Music/` — they show up in the Music app. Requires client restart.

**Images:** Drop `.png`/`.tga`/`.blp` files in `HearthPhone/Gallery/` — use them as wallpapers. Requires client restart.

## Commands

- `/phone` — Toggle phone
- `/phonereset` — Reset phone scale to default
